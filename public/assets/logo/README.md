# Logo Directory

## Required Logo Files

Please add the following logo files to this directory:

### Main Logo
- **nv_logo.png** - Main logo used throughout the application
  - Recommended size: 220px width (height auto)
  - Used in: Frontend header, footer, admin panel, email templates

### Admin Panel Logo (Optional)
If you want a different logo for the admin panel, you can update the config:
- Update `config/adminlte.php` logo settings
- Current setting points to: `assets/logo/nv_logo.png`

## Current Usage Locations

The logo is currently referenced in:
- Frontend header (70px height)
- Frontend footer (220px width)
- Admin panel brand logo
- Email templates (200px max-width)
- Invoice templates

## Instructions

1. Add your `nv_logo.png` file to this directory
2. Ensure the image is high quality and works well at different sizes
3. Consider creating different sizes if needed for optimal display
